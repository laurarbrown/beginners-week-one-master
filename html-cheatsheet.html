Radio Buttons

Radio buttons are used when you want a user to only give you one answer from multiple choices.

Each radio button can be nested within it's own Label element. By wrapping an Input element inside a Label element it will automatically associate the radio input with the label element surrounding it - if you click the text the input will be selected.

All related radio buttons should have the same name attribute to create a radio button group. By creating a group, when one button is selected, all the other buttons will be automatically deselected. Ensuring only one answer is provided by the user.

It's best practice to set a For attribute on the Label element with a value that matches the Id attribute of the Input element. This allows assistive technologies to create a linked relationship between the Label and the child Input element.

Checkboxes

Checkbox inputs are commonly used for questions that may have more than one answer.

Each checkbox can be nested with it's own Label element. As with radio buttons above.

All related checkboxes should have the same Name attribute.

Check Radio Buttons and Checkboxes by default
